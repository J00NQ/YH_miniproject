# QR 코드 기반 지능형 물류 배송 로봇 시스템 기획 초안

## 1. 프로젝트 개요

- VMware Workstation을 활용한 리눅스 20.04 및 ROS Noetic 시뮬레이션 환경에서 QR 코드 기반의 지능형 물류 배송 로봇 시스템을 개발하는 것을 목표로 합니다.
- OpenCV를 활용한 QR 코드 인식 및 경로 탐색, 그리고 로봇 제어를 통해 물류 배송을 자동화하는 시스템을 구현합니다.

## 2. 프로젝트 목표

기본 목표
- 본 프로젝트의 완성을 위한 필수 기능입니다.
1. 시뮬레이션 환경에서 QR 코드 인식 및 경로 탐색
1-1. 시뮬레이션 환경에서 사용할 맵 생성
1-2. 시뮬레이션 환경에서 QR 코드 인식
1-3. 시뮬레이션 환경에서 경로 탐색
2. 시뮬레이션 환경에서 로봇 제어를 통한 물류 배송 자동화
2-1. 시뮬레이션 환경에서 로봇 제어
2-2. 시뮬레이션 환경에서 물류 배송 자동화

추가 목표
- 추가 목표는 기본 목표를 달성한 후에 순차적으로 기능을 추가하는 방식으로 진행합니다.
1. MariaDB를 활용한 물류 배송 이력 관리
2. 웹 인터페이스를 통한 물류 배송 현황 모니터링
